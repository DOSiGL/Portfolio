Le SQL

Le SQL

Le SQL est un langage avec pour objectif de communiqer avec les bases de données. Aujourd'hui, la quasi-majorité des programmes développés est accompagnée d'une base de données, ce qui fait de ce langage une des parties clés d'un logiciel.
Il existe d'autres langages qui remplissent le même rôle, le SQL reste en revanche le plus utilisé.

Le SQL et la BPCE-SI

La base de données à commencer un prendre une part importante dans ma vie professionnelle au moment où j’ai été embauché en tant qu'apprenti architecte en base de données à la BPCE-SI.
La gestion de base de données prend une place très importante dans le monde de la banque, notamment à cause de la quantité astronomique de données et du besoin d’accéder à des informations le plus rapidement possible pour offrir la meilleure expérience cliente possible. De plus, dans une entreprise où l’argent est le produit, chaque milliseconde économisée peut représenter une énorme quantité d’argent.
Mon métier consistant à m’assurer que les requêtes SQL créées n’utilisent le moins de temps de CPU tout en faisant attention à ne pas prendre trop de stockage, me spécialiser dans l’analyse de requêtes SQL fut l’objectif principal de mon alternance dans cette entreprise.
Après six mois dans ce métier, c’est plus de 1000 requêtes étudiées et environ 25% de ces dernières modifiées, soit en réécrivant les requêtes, soit en créant ou modifiant des index des tables concernées pour gagner en performances.
Aujourd’hui, j’ai acquis une expertise suffisante pour analyser et proposer des solutions de manière autonome. Je suis capable d'administrer une base de données, de la création jusqu'à l'étude de performances des requêtes.
